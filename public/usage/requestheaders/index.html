HTMX PHP Demos

Request Headers

This example shows the different information that is available in the HTMX request.

There are two div elements here, with the id's of #div1 and #div2.

This reports on the following headers:

  • HX-Target : The id of the target element. Found using the PHP method Htmx::getTarget().
  • HX-Trigger : The id of the triggered element. Found using the PHP method Htmx::getTrigger().
  • HX-Trigger-Name : The name attribute of the triggered element. Found using the PHP method Htmx::getTriggerName().
  • HX-Current-Url : The current URL of the browser (for the HTMX request). The Found using the PHP method Htmx::getCurrentUrl().
