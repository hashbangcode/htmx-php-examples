Configuring

Here we are setting the default swap behavior to be outerHTML using a meta tag with the name "htmx-config". This will replace the entirety of the button with the result from the server.

Configuration can also be set using the following script.


htmx.config.defaultSwapStyle = 'innerHTML';

Be careful with setting this configuration item as it will change how most of your HTMX elements work.
