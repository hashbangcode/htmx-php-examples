Push Url

URL history:

The two buttons here change the URL of the page using different mechanisms that come with HTMX. This is either using the "HX-Push-Url" header from the response or the "hx-push-url" attribute to update the URL when the button is clicked.

In addition to the change of URL a "hx-swap-oob" attribute is used in the response to update the URL history with the current URL at the top of the list.

Note that this just changes the URL of the page so refreshing the page after the URL has changed will result in a 404. In a larger application you would update the URL whilst injecting more content into the page and refreshing the page would load that content.
