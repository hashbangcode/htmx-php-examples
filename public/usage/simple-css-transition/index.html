CSS Transition

Click the button to see the current time as reported by the PHP backend a CSS transition is used to animate the transition so it fades in slowly.

This works as HTMX will add classes to elements during certain events. One of which is "htmx-added", which is adding when an HTML element is added to a page.

We can set the default opacity of the "p" tag inside our result div to be 1 and set a transition of 1 second when the opacity changes. The "htmx-added" class will set the opacity of the "p" tag to be 0, and because it has a higher specificity than the base rule this hides the element. When the element is added to the page it will have this class and will be removed, which means that the opacity will suddenly be set to 1, and because we have a transition the element will fade into existence.

Here are the CSS rules used on this page.

    
#div1 p {
  opacity: 1;
  transition: opacity 1s ease-in;
}
#div1 p.htmx-added {
  opacity: 0;
}

Clicking the button again will make the element fade in again.
