HTMX PHP Demos

The hx-select attribute is used to target the ID inside the response being returned from the server.
The button on this page has the hx-select attribute set to "#response". The response is inspected to find
the element with the id that matches "response" and because the hx-swap attribute to set to outerHTML the
entire button will be replaced with this element.

This is useful as it allows us to generate responses that might have more than one HTML element
but we can extract just the data we need from the response. Everything else will be ignored.

Take a look at the server side code, it contains more than just the button clicked message.
