HTMX PHP Demos

The hx-select-oob is used to perform a normal request to a backend and then
preform another action. In this case we set the value of hx-select-oob to be
"request-output". The hx-swap means that the current button will be swapped out
for a value that comes from the response. Then, if the response contains an element
with the ID of "request-output" then this is removed from the request output and
the element on the page with that ID

This allows us to update one element and then act upon another element on the page.
