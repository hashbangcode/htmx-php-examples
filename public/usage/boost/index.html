The links on this page are "boosted" using the "hx-boost" attribute. This means that when the user clicks on the links the ajax response is replaced into the body tag.

The boosted links will also change the URL of the page.

This description div uses a "hx-preserve" attribute, which allows the boost to preserve this element when replacing the content. Key to this is returning an empty div containing the same ID as this element, which means that it will be preserved on the boosted page. Preserved elements are useful to allow navigation elements to be kept on the page, even if the response doesn't contain them.

As only the content of the body is replaced during the boost operation any scripts or styles in the header of the inner pages will not be part of the created page.

To disable boosting for certain links you can set the "hx-boost" to be "false".
