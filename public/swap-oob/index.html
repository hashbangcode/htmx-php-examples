HTMX PHP Demos

The hx-swap-oob (or swap Out Of Bounds) attribute is used to tell the response
from the server to inject itself into the page.

In this example we are sending a simple get request to the backend, but part
of the response contains a div with the following stricture.

<div id="request-output" hx-swap-oob="true"> ... </div>

This tells the response to find the div with the id of "request-output" and to
swap the content with this div.

This allows the response to update different sections of the page without the request part knowing all of that logic.
