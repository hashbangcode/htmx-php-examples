• HTMX PHP Demos

    • HTMX Usage

    • Simple get - Click a button and get a response using HTMX.
    • Simple CSS Transition - Demonstration of CSS transitions.
    • Polling - Example using trigger to make a request every 2 seconds. The server can respond with a HTTP status that turns off the requests.
    • Select - An example of using hx-select to pick the content out of the request.
    • Select out of bounds - Using hx-select-oob to target a different element on the page.
    • Select out of bounds multiple - Using hx-select-oob to fill multiple elements in the page.
    • Swap - Showing how the hx-swap attribute can be used to change how HTMX adds content to the page.
    • Swap out of bounds - An example of hx-swap-oob to push elements from the server.
    • Configuration - Example of setting HTMX configuration settings.
    • Push Url - Example of setting the URL of the page.
    • Boost - Using boost to load pages using ajax.
    • Request Headers - Reporting on the headers that are sent in a HTMX request.
