Simple Comments

A very simple comment system in a simple HTML list.

When the element is loaded into the viewport we fetch the comments from the loadcomments.php endpoint. This load loads the existing comments from a CSV file and inject them into the list.

Clicking the button will trigger a request to the savecomment.php endpoint, we use the hx-swap attribute set to outerHTML to totally replace the button with the comment form.

The submit handler of the comment form will send a post request to the same savecomment.php endpoint, which will write the comment to the CSV file and render the new comment and a new button so we can submit another comment. This the replaced the comment form, creating a couple of new list items.
