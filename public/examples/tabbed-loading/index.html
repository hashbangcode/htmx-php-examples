This shows an example of tabs without using JavaScript to control the tabs. When the user clicks on a tab label a request is made to the backend to fetch the data for that item. The response is then placed into the detail div.

The first element in the list has a "load" trigger, which means we automatically populate the tab contents from the first element when the page loads.

A small amount of JavaScript is added to the page to update which "tab" was clicked on.
