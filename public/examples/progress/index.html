Progress

Start Progress

This is an example of a progress bar that uses button to initiate the start of the progress.

This will trigger a request to the start.php callback, which will return the markup for the progress bar. Included in this is a trigger that will poll the progress.php endpoint every 600ms. The response of the progress.php endpoint is a div containing the progress bar at different states and a PHP session is used to manage the growing state of the bar.

When the progress bar reaches the maximum level we issue a "HX-Trigger" header, with the value of "done". This will trigger a custom trigger we set in the markup of the start markup, which will trigger a request to the done.php endpoint and swap the progress bar for another button so that we can restart the progress bar.

A small amount of styles is used to set up the progress bar and to animate the progress bar moving from one state to the next.
