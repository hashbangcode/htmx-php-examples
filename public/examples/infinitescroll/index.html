Infinite Scroll

This example shows the "hx-trigger" revealed event in action, which is used to fetch the next page of results from the backend.

The backend response contains another "revealed" trigger, so when it is loaded onto the page another set of results is fetched.

This will keep counting up whilst the user keeps scrolling down.
