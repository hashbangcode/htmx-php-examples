Counter

 

An old school page counter.

This increments every time you refresh the page and the counter is kept in a simple text file named after a hash of the current page URL. This means that the same element can be put on any page and the counter would increment for each page on the site.

This is not suitable for high traffic sites since we are making an uncached callback and writing to a text file.

This shows an example of delaying the callback by a 500ms before loading.

We make use of HTMX CSS transitions to fade the number in.
