Add More

Click the "Add more" button to add another input box to the list. This will issue a request and the response will contain a new text element.

Included in the response is a button containing a swap-oob attribute, which is used to piggy-back a new version of the button in the response. This updates the counter for the form elements and means that every new element gets a unique label, id, and name.
