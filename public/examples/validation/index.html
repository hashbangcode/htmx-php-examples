Form Validation

A simple form validation that will validate that a user has entered an email address into the email field.

The "hx-trigger" attribute has a value of "keyup delay:1s", which is used to trigger the event only after the user has stopped typing for 1 second. At which point the request is made to validate the email.

The response will contain the entire div element surrounding the email input field, plus an extra div telling the user that the email address is valid or not.
